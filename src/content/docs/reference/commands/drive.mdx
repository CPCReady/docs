## Usage

```bash
Usage: cpc drive [OPTIONS] COMMAND [ARGS]...

  Virtual drive management commands.

Commands:
  A       Mount DSK file in drive A
  B       Mount DSK file in drive B
  status  Display current disk status
  eject   Eject disk from drive
```

## drive A

Mount a DSK file in virtual drive A. Validates the file path and updates configuration for subsequent operations.

### Usage

```bash
Usage: cpc drive A [OPTIONS] DSK_FILE

  Mount DSK file in drive A.

Arguments:
  DSK_FILE  Path to the DSK file to mount

Options:
  --help  Show this message and exit.
```

### Examples

#### Mount disk in current directory
```bash
cpc drive A system.dsk
```

#### Mount disk with absolute path
```bash
cpc drive A /Users/user/cpc/games.dsk
```

#### Mount disk with relative path
```bash
cpc drive A ../disks/backup.dsk
```

<Aside type="caution">
If the DSK file doesn't exist, the command will display an error and won't modify the configuration.
</Aside>

<Aside type="tip">
Use `cpc drive status` to verify the disk is mounted correctly.
</Aside>

</TabItem>

## drive status

Display the current status of both floppy drives, showing which DSK files are mounted and their paths.

### Usage

```bash
Usage: cpc drive status [OPTIONS]

  Display current disk status for drives A and B.

Options:
  --help  Show this message and exit.
```

### Examples

#### Display drive status
```bash
cpc drive status
```

#### Alternative command
```bash
cpcdrivestatus
```

<Aside type="tip">
Shows which DSK files are currently mounted in drives A and B, along with their full paths.
</Aside>

<Aside type="note">
Drive configuration is stored in `~/.config/cpcready/cpcready.toml` under the `[drive]` section.
</Aside>

</TabItem>

## drive eject

Eject (unmount) the currently inserted DSK disk from drive A or B, clearing the drive configuration.

### Usage

```bash
Usage: cpc drive eject [OPTIONS]

  Eject disk from drive A or B.

Options:
  -A, --drive-a  Eject from drive A
  -B, --drive-b  Eject from drive B
  --help         Show this message and exit.
```

### Examples

#### Eject from drive A
```bash
cpc drive eject -A
```

#### Eject from drive B
```bash
cpc drive eject -B
```

#### Alternative command (drive A)
```bash
cpcdriveeject -A
```

#### Alternative command (drive B)
```bash
cpcdriveeject -B
```

<Aside type="tip">
Use `cpc drive status` after ejecting to verify the drive is empty.
</Aside>

<Aside type="note">
Ejecting removes the DSK path from `~/.config/cpcready/cpcready.toml` for the specified drive.
</Aside>
