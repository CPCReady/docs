## Uso

```bash
Uso: cpc drive [OPTIONS] COMMAND [ARGS]...

  Virtual drive management commands.

Comandos:
  A       Montar DSK file in drive A
  B       Montar DSK file in drive B
  status  Mostrar current disk status
  eject   Expulsar disco de unidad
```

## drive A

Montar a DSK file in virtual drive A. Validates the file path and updates configuration for subsequent operations.

### Uso

```bash
Uso: cpc drive A [OPTIONS] DSK_FILE

  Montar DSK file in drive A.

Argumentos:
  DSK_FILE  Path to the DSK file to mount

Opciones:
  --help  Mostrar this message and exit.
```

### Ejemplos

#### Montar disk in current directory
```bash
cpc drive A system.dsk
```

#### Montar disk with absolute path
```bash
cpc drive A /Users/user/cpc/games.dsk
```

#### Montar disk with relative path
```bash
cpc drive A ../disks/backup.dsk
```

<Aside type="caution">
Si the DSK file doesn't exist, the command will display an error and won't modify the configuration.
</Aside>

<Aside type="tip">
Use `cpc drive status` to verify the disk is mounted correctly.
</Aside>

</TabItem>

## drive status

Mostrar the current status of both floppy drives, showing which DSK files are mounted and their paths.

### Uso

```bash
Uso: cpc drive status [OPTIONS]

  Mostrar current disk status for drives A and B.

Opciones:
  --help  Mostrar this message and exit.
```

### Ejemplos

#### Mostrar drive status
```bash
cpc drive status
```

#### Alternative command
```bash
cpcdrivestatus
```

<Aside type="tip">
Shows which DSK files are currently mounted in drives A and B, along with their full paths.
</Aside>

<Aside type="note">
Drive configuration is stored in `~/.config/cpcready/cpcready.toml` under the `[drive]` section.
</Aside>

</TabItem>

<TabItem label="drive eject">

## drive eject

Expulsar (unmount) the currently inserted DSK disk from drive A o B, clearing the drive configuration.

### Uso

```bash
Uso: cpc drive eject [OPTIONS]

  Expulsar disco de unidad A o B.

Opciones:
  -A, --drive-a  Expulsar from drive A
  -B, --drive-b  Expulsar from drive B
  --help         Mostrar this message and exit.
```

### Ejemplos

#### Expulsar from drive A
```bash
cpc drive eject -A
```

#### Expulsar from drive B
```bash
cpc drive eject -B
```

#### Alternative command (drive A)
```bash
cpcdriveeject -A
```

#### Alternative command (drive B)
```bash
cpcdriveeject -B
```

<Aside type="tip">
Use `cpc drive status` after ejecting to verify the drive is empty.
</Aside>

<Aside type="note">
Ejecting removes the DSK path from `~/.config/cpcready/cpcready.toml` for the specified drive.
</Aside>
