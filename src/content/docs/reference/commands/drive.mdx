<Card>Gestionar virtual floppy drives A and B: mount DSK files, check status, and eject disks.</Card>

## Uso

```bash
Uso: cpc drive [OPTIONS] [a|b|eject|status]

  Manage disc drives.

  Actions:
    a       - Select drive A as active drive
    b       - Select drive B as active drive
    eject   - Eject disc from drive (use with -A or -B)
    status  - Show drive status (default if no action specified)

Opciones:
  -A, --drive-a  Eject disc from drive A (use with eject action)
  -B, --drive-b  Eject disc from drive B (use with eject action)
  --help         Show this message and exit.
```

### Ejemplos

```bash
cpc drive a              # Select drive A
cpc drive b              # Select drive B
cpc drive eject -A       # Eject disc from drive A
cpc drive eject -B       # Eject disc from drive B
cpc drive status         # Show drive status
cpc drive                # Show drive status (default)
```

## drive a

Select drive A as the active drive. The disc must be already inserted using `cpc disc insert`.

### Uso

```bash
cpc drive a
```

### Ejemplos

#### Select drive A
```bash
cpc drive a
```

<Aside type="caution">
If drive A has no disc inserted, the command will display an error.
</Aside>

<Aside type="tip">
Use `cpc disc insert -A <file.dsk>` first to insert a disc into drive A.
</Aside>

</TabItem>

## drive status

Mostrar the current status of both floppy drives, showing which DSK files are mounted and their paths.

### Uso

```bash
cpc drive status
# or simply
cpc drive
```

### Ejemplos

#### Mostrar drive status
```bash
cpc drive status
```

#### Default action (no arguments)
```bash
cpc drive
```

<Aside type="tip">
Shows which DSK files are currently mounted in drives A and B, along with their full paths and which drive is currently selected.
</Aside>

<Aside type="note">
Drive configuration is stored in `~/.config/cpcready/cpcready.toml` under the `[drive]` section.
</Aside>

</TabItem>

<TabItem label="drive eject">

## drive eject

Expulsar (unmount) the currently inserted DSK disk from drive A or B, clearing the drive configuration.

### Uso

```bash
cpc drive eject [OPTIONS]
```

### Opciones

```
  -A, --drive-a  Eject disc from drive A
  -B, --drive-b  Eject disc from drive B
```

### Ejemplos

#### Expulsar from drive A
```bash
cpc drive eject -A
```

#### Expulsar from drive B
```bash
cpc drive eject -B
```

<Aside type="tip">
Use `cpc drive status` after ejecting to verify the drive is empty.
</Aside>

<Aside type="note">
Ejecting removes the DSK path from `~/.config/cpcready/cpcready.toml` for the specified drive.
</Aside>

</TabItem>
