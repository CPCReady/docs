## disc new

Crear a new blank DSK disk image file with standard CPC format (40 pistas, 9 sectores por pista, 512 bytes por sector).

### Uso

```bash
Uso: cpc disc new [OPTIONS] DSK_FILE

  Crear a new blank DSK disk image.

Argumentos:
  DSK_FILE  Ruta al archivo DSK a crear

Opciones:
  --help  Mostrar this message and exit.
```

### Ejemplos

#### Crear blank disk in current directory
```bash
cpc disc new mydisk.dsk
```

#### Crear disk with full path
```bash
cpc disc new /path/to/newdisk.dsk
```

#### Alternative command
```bash
cpcdisc new blank.dsk
```

<Aside type="caution">
Si the DSK file already exists, this command will **overwrite** it without confirmation.
</Aside>

<Aside type="note">
Creates a standard CPC Data format disk: 40 tracks × 9 sectors × 512 bytes = 180KB capacity.
</Aside>

## disc insert

Insertar an existing DSK file into virtual drive A o B. The command validates file existence and updates drive configuration.

### Uso

```bash
Uso: cpc disc insert [OPTIONS] DSK_FILE

  Insertar existing DSK into drive A o B.

Argumentos:
  DSK_FILE  Path to the DSK file to insert

Opciones:
  -A, --drive-a  Insertar into drive A
  -B, --drive-b  Insertar into drive B
  --help         Mostrar this message and exit.
```

### Ejemplos

#### Insertararar disco en unidad A
```bash
cpc disc insert game.dsk -A
```

#### Insertararar disco en unidad B with relative path
```bash
cpc disc insert ../disks/system.dsk -B
```

#### Alternative command
```bash
cpcdisc insert backup.dsk -A
```

<Aside type="caution">
You must specify either `-A` or `-B`. The command will fail if:
- Neither drive option is specified
- Both drive options are specified
- The DSK file doesn't exist
</Aside>

<Aside type="tip">
Use `cpc drive status` after inserting to verify the disk is mounted correctly.
</Aside>

</TabItem>

## disc info

Mostrar detailed information about a disk image including structure, format, capacity, file types, and block usage.

### Uso

```bash
Uso: cpc disc info [OPTIONS] [DSK_FILE]

  Mostrar detailed information about a disc file.

Argumentos:
  DSK_FILE  Path to DSK file (optional if using drive option)

Opciones:
  -A, --drive-a  Mostrar info for disc in drive A
  -B, --drive-b  Mostrar info for disc in drive B
  --help         Mostrar this message and exit.
```

### Informaciónrmaciónrmation Displayed

<ul>
  <li><strong>Structure</strong>: Tracks, size, sides</li>
  <li><strong>Format</strong>: Disk type, sector information</li>
  <li><strong>Capacity</strong>: Total, used, free space</li>
  <li><strong>File Types</strong>: BASIC, Binary, ASCII counts</li>
  <li><strong>Blocks</strong>: Used blocks, first allocated block</li>
</ul>

### Ejemplos

#### Mostrar info for disk in drive A
```bash
cpc disc info -A
```

#### Mostrar info for specific DSK file
```bash
cpc disc info mydisk.dsk
```

#### Mostrar info for disk in drive B
```bash
cpc disc info -B
```

#### Alternative command
```bash
cpcdisc info -A
```

<Aside type="tip">
Without options, the command uses the default drive (usually A).
</Aside>
