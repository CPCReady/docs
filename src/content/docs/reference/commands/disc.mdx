## Usage

```bash
Usage: cpc disc [OPTIONS] COMMAND [ARGS]...

  Disk image management commands.

Commands:
  new     Create a new blank DSK disk image
  insert  Insert existing DSK into drive A or B
  info    Show detailed information about disk
```

## disc new

Create a new blank DSK disk image file with standard CPC format (40 tracks, 9 sectors per track, 512 bytes per sector).

### Usage

```bash
Usage: cpc disc new [OPTIONS] DSK_FILE

  Create a new blank DSK disk image.

Arguments:
  DSK_FILE  Path to the DSK file to create

Options:
  --help  Show this message and exit.
```

### Examples

#### Create blank disk in current directory
```bash
cpc disc new mydisk.dsk
```

#### Create disk with full path
```bash
cpc disc new /path/to/newdisk.dsk
```

#### Alternative command
```bash
cpcdisc new blank.dsk
```

<Aside type="caution">
If the DSK file already exists, this command will **overwrite** it without confirmation.
</Aside>

<Aside type="note">
Creates a standard CPC Data format disk: 40 tracks × 9 sectors × 512 bytes = 180KB capacity.
</Aside>

## disc insert

Insert an existing DSK file into virtual drive A or B. The command validates file existence and updates drive configuration.

### Usage

```bash
Usage: cpc disc insert [OPTIONS] DSK_FILE

  Insert existing DSK into drive A or B.

Arguments:
  DSK_FILE  Path to the DSK file to insert

Options:
  -A, --drive-a  Insert into drive A
  -B, --drive-b  Insert into drive B
  --help         Show this message and exit.
```

### Examples

#### Insert disk into drive A
```bash
cpc disc insert game.dsk -A
```

#### Insert disk into drive B with relative path
```bash
cpc disc insert ../disks/system.dsk -B
```

#### Alternative command
```bash
cpcdisc insert backup.dsk -A
```

<Aside type="caution">
You must specify either `-A` or `-B`. The command will fail if:
- Neither drive option is specified
- Both drive options are specified
- The DSK file doesn't exist
</Aside>

<Aside type="tip">
Use `cpc drive status` after inserting to verify the disk is mounted correctly.
</Aside>

</TabItem>

## disc info

Display detailed information about a disk image including structure, format, capacity, file types, and block usage.

### Usage

```bash
Usage: cpc disc info [OPTIONS] [DSK_FILE]

  Show detailed information about a disc file.

Arguments:
  DSK_FILE  Path to DSK file (optional if using drive option)

Options:
  -A, --drive-a  Show info for disc in drive A
  -B, --drive-b  Show info for disc in drive B
  --help         Show this message and exit.
```

### Information Displayed

<ul>
  <li><strong>Structure</strong>: Tracks, size, sides</li>
  <li><strong>Format</strong>: Disk type, sector information</li>
  <li><strong>Capacity</strong>: Total, used, free space</li>
  <li><strong>File Types</strong>: BASIC, Binary, ASCII counts</li>
  <li><strong>Blocks</strong>: Used blocks, first allocated block</li>
</ul>

### Examples

#### Show info for disk in drive A
```bash
cpc disc info -A
```

#### Show info for specific DSK file
```bash
cpc disc info mydisk.dsk
```

#### Show info for disk in drive B
```bash
cpc disc info -B
```

#### Alternative command
```bash
cpcdisc info -A
```

<Aside type="tip">
Without options, the command uses the default drive (usually A).
</Aside>
