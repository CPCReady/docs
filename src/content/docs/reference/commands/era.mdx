<Card>Eliminar (erase) a file from the currently mounted DSK disk. Soporta wildcards for batch deletion.</Card>

## Uso

```bash
Uso: cpc era [OPTIONS] PATTERN

  Eliminar file(s) from DSK disk.

Argumentos:
  PATTERN  File pattern to delete (soportes wildcards: *, ?)

Opciones:
  -A, --drive-a  Eliminar from drive A (default)
  -B, --drive-b  Eliminar from drive B
  --help         Mostrar this message and exit.
```

## Ejemplos

### Eliminar specific file
```bash
cpc era GAME.BAS
```

### Eliminar all .BAK files
```bash
cpc era *.BAK
```

### Eliminar from drive B
```bash
cpc era OLDFILE.BIN -B
```

### Eliminar all files starting with TEST
```bash
cpc era TEST*.*
```

### Alternative command
```bash
cpcera MYFILE.TXT
```

<Aside type="caution">
**Deletion is permanent!** Files are removed from the DSK without confirmation. Make backups before using wildcards.
</Aside>

<Aside type="tip">
Wildcards:  
**\*** matches any characters  
**?** matches single character  
Ejemplo: `DATA?.BIN` matches DATA1.BIN, DATA2.BIN, etc.
</Aside>
