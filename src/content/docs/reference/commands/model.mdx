## Uso

```bash
Uso: cpc model [OPTIONS] [MODEL_TYPE]

  Set or display CPC model configuration.

Argumentos:
  MODEL_TYPE  CPC model: 464, 664, or 6128

Opciones:
  --help  Mostrar this message and exit.
```

## Ejemplos

### Mostrar current model
```bash
cpc model
```

### Set model to CPC 464
```bash
cpc model 464
```

### Set model to CPC 664
```bash
cpc model 664
```

### Set model to CPC 6128
```bash
cpc model 6128
```
