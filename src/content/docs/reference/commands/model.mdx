<Card>Configure or display the Amstrad CPC model to emulate. 
Supports <Badge text="464" variant="note" />, <Badge text="664" variant="note" />, and <Badge text="6128" variant="note" /> models.</Card>

## Uso

```bash
Usage: cpc model [OPTIONS] [MODEL_TYPE]

  Set or display CPC model configuration.

Arguments:
  MODEL_TYPE  CPC model: 464, 664, or 6128

Options:
  --help  Show this message and exit.
```

## Examples

### Display current model
```bash
cpc model
```

### Set model to CPC 464
```bash
cpc model 464
```

### Set model to CPC 664
```bash
cpc model 664
```

### Set model to CPC 6128
```bash
cpc model 6128
```
