---
title: user - Establecer número de usuario CP/M (0-15)
description: Gestionar áreas de usuario CP/M
---

## Sinopsis

```bash
cpc user [numero_usuario]
```

## Descripción

Establece o consulta el número de usuario CP/M actual (0-15) usado al guardar archivos.

## Ejemplos

```bash
cpc user 3
cpc user
```

## Notas

- El valor se persiste en la configuración y es usado por los comandos save/cat.
