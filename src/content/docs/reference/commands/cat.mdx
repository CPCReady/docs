<Card>Lista los archivos del disco virtual seleccionado. Sin opciones se usa la unidad por defecto. Usa `-A` o `-B` para especificar la unidad.</Card>

## Uso

```bash
Usage: cpc cat [OPTIONS]

  List files in the virtual disc. If no option is selected, the selected drive
  is displayed.

Options:
  -A, --drive-a  List files in the virtual disc from drive A
  -B, --drive-b  List files in the virtual disc from drive B
  --version      Show version and exit
  --help         Show this message and exit.
```

```bash
Usage: catcpc [OPTIONS]

  List files in the virtual disc. If no option is selected, the selected drive
  is displayed.

Options:
  -A, --drive-a  List files in the virtual disc from drive A
  -B, --drive-b  List files in the virtual disc from drive B
  --version      Show version and exit
  --help         Show this message and exit.
```

## Notas

- Muestra nombre, tipo, tamaño y área de usuario CP/M.
- Si no hay disco montado se muestra un error.
