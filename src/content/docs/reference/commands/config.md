---
title: config - Configuración en terminal
description: Configura unidades y ajustes a través de una UI en terminal (npyscreen)
---

## Sinopsis

```bash
cpc config
```

## Descripción

Abre una interfaz en la terminal para configurar rutas de unidades, ajustes de M4Board y otras preferencias.

## Ejemplos

```bash
cpc config
```

## Notas

- Usa `npyscreen` para formularios interactivos.
- Guarda la configuración en el directorio de configuración del usuario.
