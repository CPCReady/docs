<Card>Mostrar all CPCReady configuration settings from TOML file including drive mappings, emulator path, CPC model, and modo de vídeo.</Card>

## Uso

```bash
Uso: cpc settings [OPTIONS]

  Mostrar toda la configuración settings.

Opciones:
  --help  Mostrar this message and exit.
```

## Ejemplos

### Mostrar all settings
```bash
cpc settings
```

### Alternative command
```bash
cpcsettings
```

<Aside type="tip">
Use this command to verify your configuration after making changes with `cpc config` or manually editing the TOML file.
</Aside>

<Aside type="note">
Configuración file location: `~/.config/cpcready/cpcready.toml`  
Displays: drive A/B paths, emulator path, CPC model, modo de vídeo, y ajustes del sistema.
</Aside>
