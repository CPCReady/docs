<Card>Configurar or display the CP/M user number (0-15). User numbers allow organizing files into separate areas on the same disk.</Card>

## Uso

```bash
Uso: cpc user [OPTIONS] [USER_NUMBER]

  Set or display current user number.

Argumentos:
  USER_NUMBER  User number (0-15)

Opciones:
  --help  Mostrar this message and exit.
```

## Ejemplos

### Mostrar current user
```bash
cpc user
```

### Set user to 0 (default)
```bash
cpc user 0
```

### Set user to 5
```bash
cpc user 5
```

### Set user to 15 (maximum)
```bash
cpc user 15
```
