<Card>Configure or display the CP/M user number (0-15). User numbers allow organizing files into separate areas on the same disk.</Card>

## Usage

```bash
Usage: cpc user [OPTIONS] [USER_NUMBER]

  Set or display current user number.

Arguments:
  USER_NUMBER  User number (0-15)

Options:
  --help  Show this message and exit.
```

## Examples

### Display current user
```bash
cpc user
```

### Set user to 0 (default)
```bash
cpc user 0
```

### Set user to 5
```bash
cpc user 5
```

### Set user to 15 (maximum)
```bash
cpc user 15
```

<Aside type="tip">
User 0 is the default area. Use different user numbers to organize files (e.g., user 1 for games, user 2 for utilities).
</Aside>

<Aside type="note">
Valid range: 0-15  
Configuration stored in `~/.config/cpcready/cpcready.toml`  
Commands like `cpc list` and `cpc cat` respect the current user number.
</Aside>
