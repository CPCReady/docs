<Card>Rename a file on a DSK disk image. The file must exist on the selected disk and the new name must not already exist.
Use <Badge text="-A" variant="note" /> or <Badge text="-B" variant="note" /> to specify the drive.</Card>

## Uso

```bash
Usage: cpc ren [OPTIONS] OLD_NAME NEW_NAME

  Rename a file on the disc.

Arguments:
  OLD_NAME  Current file name  [required]
  NEW_NAME  New file name      [required]

Options:
  -A, --drive-a  Rename file on drive A
  -B, --drive-b  Rename file on drive B
  --help         Show this message and exit.
```

## Examples

### Rename a file on default drive
```bash
cpc ren OLDGAME.BAS NEWGAME.BAS
```

### Rename on drive A
```bash
cpc ren MENU.BIN INTRO.BIN -A
```

### Rename on drive B
```bash
cpc ren DATA.TXT CONFIG.TXT -B
```
