<Card>Launch the <Badge text="CPCReadyConfig" variant="success" /> graphical interface for easy configuration of drives, emulator, CPC model, and system settings.</Card>

## Usage

```bash
Usage: cpc configweb [OPTIONS]

  Launch graphical configuration interface.

Options:
  --help  Show this message and exit.
```

## Examples

### Launch config GUI
```bash
cpc configweb
```

### Alternative command
```bash
cpcconfigweb
```

<Aside type="tip">
The GUI provides visual controls for:  
• Drive A/B DSK file selection  
• Emulator path configuration  
• CPC model (464/664/6128) selection  
• Video mode (0/1/2) settings  
• User number configuration
</Aside>

<Aside type="note">
Built with **PySide6** (Qt6). All changes are saved to `~/.config/cpcready/cpcready.toml`.
</Aside>
