<Card>Lanza the <Badge text="CPCReadyConfig" variant="success" /> graphical interface for easy configuration of drives, emulator, CPC model, y ajustes del sistema.</Card>

## Uso

```bash
Uso: cpc configweb [OPTIONS]

  Lanza graphical configuration interface.

Opciones:
  --help  Mostrar this message and exit.
```

## Ejemplos

### Lanza config GUI
```bash
cpc configweb
```

### Alternative command
```bash
cpcconfigweb
```

<Aside type="tip">
The GUI provides visual controls for:  
• Drive A/B DSK file selection  
• Emulator path configuration  
• CPC model (464/664/6128) selection  
• Video mode (0/1/2) settings  
• User number configuration
</Aside>

<Aside type="note">
Built with **PySide6** (Qt6). All changes are saved to `~/.config/cpcready/cpcready.toml`.
</Aside>
