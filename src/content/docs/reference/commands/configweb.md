---
title: configweb - Configuración web (Streamlit)
description: Inicia la UI web en Streamlit para configurar unidades y ajustes de M4Board
---

## Sinopsis

```bash
cpc configweb
```

## Descripción

Inicia una aplicación Streamlit que expone las mismas opciones de configuración mediante una interfaz web. La app oculta el menú de Streamlit y muestra el logo del proyecto y la versión.

## Ejemplos

```bash
cpc configweb
```

## Notas

- Requiere tener `streamlit` instalado.
- Valida direcciones IP para opciones relacionadas con M4Board.
