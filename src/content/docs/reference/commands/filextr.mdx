<Card>Extraer one or more files from a DSK disk image to the local file system. 
Use <Badge text="-A" variant="note" /> or <Badge text="-B" variant="note" /> to specify the source drive.</Card>

## Uso

```bash
Uso: cpc filextr [OPTIONS] FILES...

  Extraer files from disc.

Argumentos:
  FILES...  Names of files to extract from the disk  [required]

Opciones:
  -A, --drive-a  Extraer from drive A
  -B, --drive-b  Extraer from drive B
  --help         Mostrar this message and exit.
```

## Ejemplos

### Extraer a single file
```bash
cpc filextr GAME.BAS
```

### Extraer multiple files
```bash
cpc filextr MENU.BIN LOADER.BAS DATA.BIN
```

### Extraer from drive A
```bash
cpc filextr GAME.BAS -A
```

### Extraer from drive B
```bash
cpc filextr *.BAS -B
```
