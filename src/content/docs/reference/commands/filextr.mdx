<Card>Extract one or more files from a DSK disk image to the local file system. 
Use <Badge text="-A" variant="note" /> or <Badge text="-B" variant="note" /> to specify the source drive.</Card>

## Uso

```bash
Usage: cpc filextr [OPTIONS] FILES...

  Extract files from disc.

Arguments:
  FILES...  Names of files to extract from the disk  [required]

Options:
  -A, --drive-a  Extract from drive A
  -B, --drive-b  Extract from drive B
  --help         Show this message and exit.
```

## Examples

### Extract a single file
```bash
cpc filextr GAME.BAS
```

### Extract multiple files
```bash
cpc filextr MENU.BIN LOADER.BAS DATA.BIN
```

### Extract from drive A
```bash
cpc filextr GAME.BAS -A
```

### Extract from drive B
```bash
cpc filextr *.BAS -B
```
