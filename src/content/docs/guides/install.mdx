## Descripción General

**CPCReady** es una herramienta basada en Python para gestionar imágenes de disco Amstrad CPC. Sigue esta guía para instalarla en tu sistema.

<Aside type="note">
CPCReady funciona en **macOS**, **Linux** y **Windows**. Python 3.13+ requerido.
</Aside>

---

## Instalación Rápida

<Tabs>
  <TabItem label="macOS / Linux">
  
  ### Homebrew (Recomendado)
  
  ```bash
  # Agregar el tap de CPCReady
  brew tap cpcready/cpcready https://github.com/cpcready/homebrew-cpcready
  
  # Instalar cpc
  brew install cpcready/cpcready/cpc
  
  # Verificar instalación
  cpc --version
  ```
  
  <Aside type="tip">
  Si no tienes Homebrew instalado, sigue las instrucciones en la sección "Instalar Gestores de Paquetes" más abajo.
  </Aside>
  
  </TabItem>
  
  <TabItem label="Windows">
  
  ### Chocolatey (Recomendado)
  
  ```powershell
  # Instalar cpc
  choco install cpc
  
  # Verificar instalación
  cpc --version
  ```
  
  <Aside type="tip">
  Si no tienes Chocolatey instalado, sigue las instrucciones en la sección "Instalar Gestores de Paquetes" más abajo.
  </Aside>
  
  </TabItem>
</Tabs>

## Instalar Gestores de Paquetes

### Homebrew (macOS / Linux)

Si aún no tienes Homebrew instalado:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

**En Linux**, después de la instalación, agrega Homebrew a tu PATH:

```bash
echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> ~/.bashrc
source ~/.bashrc
```

Más información: [brew.sh](https://brew.sh)

### Chocolatey (Windows)

Si aún no tienes Chocolatey instalado:

1. Abre PowerShell como **Administrador**
2. Ejecuta:

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
```

3. Cierra y vuelve a abrir PowerShell

Más información: [chocolatey.org](https://chocolatey.org/install)

## Métodos de Instalación Alternativos

<Aside type="note">
Si ya instalaste CPCReady con Homebrew o Chocolatey, puedes saltar esta sección.
</Aside>

<Tabs>
  <TabItem label="Desde Código Fuente">

### Instalar desde Repositorio GitHub

<Aside type="caution">
Este método requiere tener **Python 3.13+** y **Poetry** instalados previamente.
</Aside>

<Steps>

1. **Clonar el repositorio**
   
   ```bash
   git clone https://github.com/CPCReady/cpc.git
   cd cpc
   ```

2. **Instalar con Poetry**
   
   ```bash
   poetry install
   ```

3. **Verificar instalación**
   
   ```bash
   poetry run cpc --version
   ```

4. **Opcional: Añadir al PATH**
   
   Para un acceso más fácil, puedes crear un alias en tu shell:
   
   ```bash
   # Para bash/zsh, añadir a ~/.bashrc o ~/.zshrc
   alias cpc='poetry run cpc'
   ```

</Steps>

<Aside type="tip">
Instalar desde código fuente te da las últimas características y te permite contribuir al desarrollo.
</Aside>

  </TabItem>
  
  <TabItem label="Con pip (Avanzado)">

### Instalar con pip

Si prefieres usar pip directamente:

```bash
# Clonar el repositorio
git clone https://github.com/CPCReady/cpc.git
cd cpc

# Instalar con pip
pip install .

# Verificar instalación
cpc --version
```

<Aside type="caution">
Asegúrate de usar Python 3.13+ y tener pip actualizado: `pip install --upgrade pip`
</Aside>

  </TabItem>
</Tabs>

## Herramientas Requeridas para Instalación desde Código

<Aside type="note">
Solo necesario si instalas desde código fuente. Si usas Homebrew/Chocolatey, estos gestores manejan las dependencias automáticamente.
</Aside>

### 1. Configurar CPCReady

Ejecutar el asistente de configuración:

```bash
poetry run cpc configweb
```

O editar manualmente el archivo de configuración:

**Ubicación**: `~/.config/cpcready/cpcready.toml`

```toml
[drive]
drive_a = ""
drive_b = ""
selected_drive = "A"

[emulator]
default = "RetroVirtualMachine"
retro_virtual_machine_path = "/Applications/Retro Virtual Machine 2.app"

[system]
user = 0
model = "6128"
mode = 1
```

### 2. Configurar Emulador (Opcional)

Si quieres usar el `cpc run` comando, configura la ruta de tu emulador:

<Steps>

1. Descargar **RetroVirtualMachine** desde [retrovirtualmachine.org](https://www.retrovirtualmachine.org/)

2. Instala el emulador en tu sistema

3. Configura la ruta en CPCReady:
   
   ```bash
   poetry run cpc configweb
   ```
   
   O editar `~/.config/cpcready/cpcready.toml` directamente:
   
   <Tabs>
     <TabItem label="macOS">
     ```toml
     [emulator]
     retro_virtual_machine_path = "/Applications/Retro Virtual Machine 2.app"
     ```
     </TabItem>
     
     <TabItem label="Linux">
     ```toml
     [emulator]
     retro_virtual_machine_path = "/usr/local/bin/RetroVirtualMachine"
     ```
     </TabItem>
     
     <TabItem label="Windows (WSL)">
     ```toml
     [emulator]
     retro_virtual_machine_path = "/mnt/c/Program Files/RetroVirtualMachine/rvm.exe"
     ```
     </TabItem>
   </Tabs>

</Steps>

## Verificar Instalación

Prueba tu instalación con estos comandos:

<Tabs>
  <TabItem label="Homebrew / Chocolatey">
  
  ```bash
  # Verificar versión
  cpc --version
  
  # Ver ayuda
  cpc --help
  
  # Crear un disco de prueba
  cpc disc new test.dsk
  
  # Montar el disco
  cpc drive A test.dsk
  
  # Listar archivos (debería estar vacío)
  cpc cat
  ```
  
  </TabItem>
  
  <TabItem label="Código Fuente">
  
  ```bash
  # Verificar versión
  poetry run cpc --version
  
  # Ver ayuda
  poetry run cpc --help
  
  # Crear un disco de prueba
  poetry run cpc disc new test.dsk
  
  # Montar el disco
  poetry run cpc drive A test.dsk
  
  # Listar archivos (debería estar vacío)
  poetry run cpc cat
  ```
  
  </TabItem>
</Tabs>

<Aside type="tip" title="¡Éxito!">
Si todos los comandos se ejecutan sin errores, CPCReady está correctamente instalado y listo para usar.
</Aside>

## Solución de Problemas

### Homebrew: comando no encontrado

Después de instalar Homebrew, asegúrate de agregarlo a tu PATH:

**macOS:**
```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc
```

**Linux:**
```bash
echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> ~/.bashrc
source ~/.bashrc
```

### Chocolatey: comando no encontrado

Después de instalar Chocolatey, reinicia PowerShell o tu terminal.

Si persiste el problema, verifica que Chocolatey esté en tu PATH:

```powershell
$env:Path
```

### Python: versión incorrecta

CPCReady requiere Python 3.13+. Verifica tu versión:

```bash
python3 --version
```

Si necesitas actualizar Python:
- **macOS/Linux con Homebrew**: `brew install python@3.13`
- **Windows con Chocolatey**: `choco install python --version=3.13.0`
- **Descarga directa**: [python.org](https://www.python.org/downloads/)

### Problemas de permisos (macOS/Linux)

Si encuentras errores de permisos al instalar con Homebrew:

```bash
sudo chown -R $(whoami) $(brew --prefix)/*
```
