### Commands Introduced

**Disk & Drive Management:**
- `cpc disc new` - Create blank DSK image
- `cpc disc insert` - Insert disk into drive
- `cpc disc info` - Display disk information
- `cpc drive A/B` - Mount disk in drive
- `cpc drive status` - Show current drive status
- `cpc drive eject` - Eject disk from drive

**File Management:**
- `cpc cat` - List all files on disk
- `cpc save` - Save file to disk (auto/binary/program/ASCII)
- `cpc filextr` - Extract file from disk
- `cpc list` - Display BASIC program (detokenized)
- `cpc era` - Delete file(s) with wildcards
- `cpc ren` - Rename file on disk

**System Configuration:**
- `cpc model` - Set CPC model (464/664/6128)
- `cpc mode` - Set video mode (0/1/2)
- `cpc user` - Set CP/M user number (0-15)
- `cpc settings` - Display all configuration
- `cpc configweb` - Launch GUI configuration tool

**Emulator:**
- `cpc run` - Launch RetroVirtualMachine with disk

### Technical Details

**Core Technologies:**
- Python 3.13+
- Poetry for dependency management
- Click 8.3.0 for CLI framework
- PySide6 6.10.0 for GUI
- TOML configuration format
- Custom DSK format library (pydsk)

**File Type Support:**
- Binary files with AMSDOS headers (type 'b')
- BASIC programs (tokenized and ASCII)
- ASCII/text files (type 'a')
- Automatic type detection
- Hex (0x/&) and decimal address support

**Configuration System:**
- Location: `~/.config/cpcready/cpcready.toml`
- Sections: `[drive]`, `[emulator]`, `[system]`
- Automatic creation on first run
- GUI editor with PySide6
