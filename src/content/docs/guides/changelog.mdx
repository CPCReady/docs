### Comandos Introduced

**Disk & Gestión de Unidades:**
- `cpc disc new` - Crear imagen DSK vacía
- `cpc disc insert` - Insertar disco en unidad
- `cpc disc info` - Mostrar información del disco
- `cpc drive A/B` - Montar disco en unidad
- `cpc drive status` - Mostrar estado actual de unidad
- `cpc drive eject` - Expulsar disco de unidad

**Gestión de Archivos:**
- `cpc cat` - Listar todos los archivos en disco
- `cpc save` - Guardar archivo en disco (auto/binary/program/ASCII)
- `cpc filextr` - Extraer archivo del disco
- `cpc list` - Mostrar BASIC program (detokenized)
- `cpc era` - Eliminar archivo(s) con comodines
- `cpc ren` - Renombrar archivo en disco

**System Configuración:**
- `cpc model` - Configurar modelo CPC (464/664/6128)
- `cpc mode` - Set modo de vídeo (0/1/2)
- `cpc user` - Configurar número de usuario CP/M (0-15)
- `cpc settings` - Mostrar toda la configuración
- `cpc configweb` - Lanza Herramienta de configuración GUI

**Emulador:**
- `cpc run` - Lanza RetroVirtualMachine with disk

### Detalles Técnicos

**Tecnologías Principales:**
- Python 3.13+
- Poetry para gestión de dependencias
- Click 8.3.0 para framework CLI
- PySide6 6.10.0 para GUI
- TOML formato de configuración
- Biblioteca personalizada de formato DSK (pydsk)

**Soporte de Tipos de Archivo:**
- Archivos binarios con cabeceras AMSDOS (type 'b')
- BASIC programs (tokenized and ASCII)
- Archivos ASCII/texto (type 'a')
- Detección automática de tipo
- Hex (0x/&) and decimal address soporte

**Configuración System:**
- Ubicación: `~/.config/cpcready/cpcready.toml`
- Secciones: `[drive]`, `[emulator]`, `[system]`
- Creación automática en primera ejecución
- Editor GUI con PySide6
