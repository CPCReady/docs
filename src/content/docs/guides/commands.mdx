## Descripción General de Comandos

CPCReady proporciona un conjunto completo de comandos que ayundan a trabajar con imagenes de disco y cinta para programacion en BASIC. .

  <Aside type="tip" title="Nota">
  Se aceptan todos los comandos de CPCReady a través del CLI <Badge text="cpc" variant="note" />. Tambien puedes ejecutar comandos individuales directamente desde la terminal con el prefijo <Badge text="cpc-xxxx" variant="note" />
</Aside>

## Referencia Rápida de Comandos

### Comandos de Disco y Unidad

| Comando | Descripción | Ejemplo |
|---------|-------------|---------|  
| `cpc disc new <file>` | Crear DSK vacío | `cpc disc new blank.dsk` |
| `cpc disc insert <file> -A` | Insertar disco en unidad A | `cpc disc insert game.dsk -A` |
| `cpc disc info -A` | Mostrar información del disco | `cpc disc info -A` |
| `cpc drive A <file>` | Montar disco en unidad A | `cpc drive A system.dsk` |
| `cpc drive B <file>` | Montar disco en unidad B | `cpc drive B games.dsk` |
| `cpc drive status` | Mostrar estado de unidades | `cpc drive status` |
| `cpc drive eject -A` | Expulsar de unidad A | `cpc drive eject -A` |

### Comandos de Gestión de Archivos

| Comando | Descripción | Ejemplo |
|---------|-------------|---------|  
| `cpc cat` | Listar archivos en disco | `cpc cat` |
| `cpc save <file> [type]` | Guardar archivo en disco | `cpc save loader.bin b 0x4000` |
| `cpc filextr <file>` | Extraer archivo del disco | `cpc filextr GAME.BAS` |
| `cpc list <file>` | Mostrar programa BASIC | `cpc list MENU.BAS` |
| `cpc era <file>` | Eliminar archivo | `cpc era OLDFILE.BIN` |
| `cpc ren <old> <new>` | Renombrar archivo | `cpc ren OLD.BAS NEW.BAS` |

### Comandos del Sistema

| Comando | Descripción | Ejemplo |
|---------|-------------|---------|  
| `cpc model <464\|664\|6128>` | Configurar modelo CPC | `cpc model 6128` |
| `cpc mode <0\|1\|2>` | Configurar modo de vídeo | `cpc mode 1` |
| `cpc user <0-15>` | Configurar usuario CP/M | `cpc user 0` |
| `cpc run [file]` | Lanzar emulador | `cpc run GAME.BAS` |---

## Obtener Ayuda

Para ayuda detallada sobre cualquier comando, usa la bandera `--help`:

```bash
cpc --help              # Ayuda general
cpc disc --help         # Ayuda para el comando disc
cpc save --help         # Ayuda para el comando save
```

## Notificaciones de Actualización

CPCReady incluye un sistema automático de verificación de actualizaciones que funciona de forma transparente con todos los comandos.

### ¿Cómo Funciona?

Cuando ejecutes cualquier comando principal de CPCReady (como `cpc`, `disc`, `cat`, `drive`, etc.), la herramienta verificará automáticamente si hay una nueva versión disponible en PyPI.

### Ejemplo de Notificación

Si hay una actualización disponible, verás un mensaje como este antes de la salida normal del comando:

```bash
⚠ New version available: v3.1.0 (current: v3.0.1)
⚠ Update with: pipx upgrade cpcready

# Seguido de la salida normal del comando
```

### Características del Sistema

- **Cache inteligente**: Las consultas se almacenan en caché durante 6 horas para evitar solicitudes excesivas
- **No intrusivo**: Si no hay conexión a internet, falla silenciosamente sin afectar el comando
- **Timeout rápido**: Máximo 3 segundos de espera para la verificación
- **Automático**: No requiere configuración adicional

<Aside type="tip">
Las notificaciones solo aparecen cuando hay una versión más reciente disponible. Si estás ejecutando la última versión, no verás ningún mensaje.
</Aside>
