## Command Overview

CPCReady provides a comprehensive set of commands for managing Amstrad CPC disk images, files, and emulator integration. All commands are accessible through the `cpc` CLI.

## Quick Command Reference

### Disk & Drive Commands

| Command | Description | Example |
|---------|-------------|---------|
| `cpc disc new <file>` | Create blank DSK | `cpc disc new blank.dsk` |
| `cpc disc insert <file> -A` | Insert disk in drive A | `cpc disc insert game.dsk -A` |
| `cpc disc info -A` | Show disk information | `cpc disc info -A` |
| `cpc drive A <file>` | Mount disk in drive A | `cpc drive A system.dsk` |
| `cpc drive B <file>` | Mount disk in drive B | `cpc drive B games.dsk` |
| `cpc drive status` | Show drive status | `cpc drive status` |
| `cpc drive eject -A` | Eject from drive A | `cpc drive eject -A` |

### File Management Commands

| Command | Description | Example |
|---------|-------------|---------|
| `cpc cat` | List files on disk | `cpc cat` |
| `cpc save <file> [type]` | Save file to disk | `cpc save loader.bin b 0x4000` |
| `cpc filextr <file>` | Extract file from disk | `cpc filextr GAME.BAS` |
| `cpc list <file>` | Display BASIC program | `cpc list MENU.BAS` |
| `cpc era <file>` | Delete file | `cpc era OLDFILE.BIN` |
| `cpc ren <old> <new>` | Rename file | `cpc ren OLD.BAS NEW.BAS` |

### System Commands

| Command | Description | Example |
|---------|-------------|---------|
| `cpc model <464\|664\|6128>` | Set CPC model | `cpc model 6128` |
| `cpc mode <0\|1\|2>` | Set video mode | `cpc mode 1` |
| `cpc user <0-15>` | Set CP/M user | `cpc user 0` |
| `cpc settings` | Show all settings | `cpc settings` |
| `cpc configweb` | Launch GUI config | `cpc configweb` |
| `cpc run [file]` | Launch emulator | `cpc run GAME.BAS` |

## Getting Help

For detailed help on any command, use the `--help` flag:

```bash
cpc --help              # General help
cpc disc --help         # Help for disc command
cpc save --help         # Help for save command
```
