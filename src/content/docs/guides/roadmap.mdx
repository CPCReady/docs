## Project Vision

CPCReady aims to be the **most comprehensive and user-friendly toolchain** for Amstrad CPC development, bringing modern development workflows to vintage computing enthusiasts.

<Aside type="note">
This roadmap is a living document and subject to change based on community feedback and development priorities.
</Aside>

## Community Requests

<Aside type="tip" title="Have an Idea?">
We welcome community feedback! Submit feature requests on [GitHub Issues](https://github.com/CPCReady/cpc/issues).
</Aside>

### Most Requested Features

<Steps>

1. **CDT/Tape Support**
   - Full CDT file handling
   - Tape to disk conversion
   - Turbo loader support

2. **Snapshot Support**
   - SNA file handling
   - Save states
   - Quick load/save

3. **Graphics Tools**
   - Screen converter
   - Sprite editor
   - Tile map tools
   - Color palette editor

4. **Sound Tools**
   - AY music converter
   - Sound effect tools
   - Music tracker integration

</Steps>
