# Listado de Comandos

Puedes ejecutar **CPCReady** desde el shell con el comando `cpc`. A continuación se listan comandos relevantes y su propósito; consulta las páginas individuales en "Reference / Commands" para más detalle.

- `build` — Crear imágenes DSK/CDT y artefactos para M4 Board.
- `disc` — Crear/insertar/inspeccionar imágenes `.DSK`.
- `drive` — Montar/expulsar/ver estado de las unidades A/B.
- `cat` — Mostrar catálogo de un disco virtual.
- `save` — Guardar archivos en una imagen `.DSK`.
- `list` — Mostrar o desensamblar archivos.
- `era` — Borrar archivos usando comodines CP/M.
- `user` — Establecer número de usuario CP/M.
