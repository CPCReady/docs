<Card>
Renombrar ficheros de un fichero en una imagen de disco DSK. El archivo debe existir en el disco seleccionado y el nuevo nombre no debe existir ya.
Use <Badge text="-A" variant="note" /> or <Badge text="-B" variant="note" /> para especificar la unidad. Si no se especifica una unidad, se utiliza la unidad actualmente seleccionada.
<Aside type="tip">
La unidad seleccionada se configura con <Badge text="cpc drive A" variant="note" /> o <Badge text="cpc drive B" variant="note" />. Por defecto es la unidad A.
</Aside>
</Card>

## Uso
```bash
Uso: cpc ren [OPTIONS] OLD_NAME NEW_NAME

  Renombrar a file on the disc.

Argumentos:
  OLD_NAME  Current file name  [required]
  NEW_NAME  New file name      [required]

Opciones:
  -A, --drive-a  Renombrar file on drive A
  -B, --drive-b  Renombrar file on drive B
  --help         Mostrar this message and exit.
```

## Ejemplos

### Renombrar un fichero en la unidad por defecto
```bash
cpc ren OLDGAME.BAS NEWGAME.BAS
```

### Renombrar en la unidad A
```bash
cpc ren MENU.BIN INTRO.BIN -A
```

### Renombrar en la unidad B
```bash
cpc ren DATA.TXT CONFIG.TXT -B
```


<Aside type="caution">
Los nombres de archivo deben seguir las convenciones de nomenclatura AMSDOS (formato 8.3).
</Aside>
