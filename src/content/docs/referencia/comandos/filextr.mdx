<Card>
Extrae uno o más archivos de una imagen de disco DSK al sistema de archivos local. Use <Badge text="-A" variant="note" /> or <Badge text="-B" variant="note" /> para especificar la unidad. Si no se especifica una unidad, se utiliza la unidad actualmente seleccionada.
<Aside type="tip">
La unidad seleccionada se configura con <Badge text="cpc drive A" variant="note" /> o <Badge text="cpc drive B" variant="note" />. Por defecto es la unidad A.
</Aside>
</Card>


## Uso

```bash
Uso: cpc filextr [OPTIONS] FILES...

  Extraer files from disc.

Argumentos:
  FILES...  Names of files to extract from the disk  [required]

Opciones:
  -A, --drive-a  Extraer from drive A
  -B, --drive-b  Extraer from drive B
  --help         Mostrar this message and exit.
```

## Ejemplos

### Extraer un fichero específico
```bash
cpc filextr GAME.BAS
```

### Extraer múltiples ficheros
```bash
cpc filextr MENU.BIN LOADER.BAS DATA.BIN
```

### Extraer de drive A
```bash
cpc filextr GAME.BAS -A
```

### Extraer de drive B
```bash
cpc filextr *.BAS -B
```

<Aside type="tip">
Los archivos se extraen al directorio de trabajo actual con sus nombres originales.
</Aside>

<Aside type="caution">
Si un archivo con el mismo nombre existe localmente, se sobrescribirá sin confirmación.
</Aside>
