<Card>Gestiona unidades de disquete virtuales A y B: monta archivos DSK, comprueba el estado y expulsa discos.</Card>

## Uso

```bash
Uso: cpc drive [OPTIONS] [a|b|eject|status]

  Manage disc drives.

  Actions:
    a       - Select drive A as active drive
    b       - Select drive B as active drive
    eject   - Eject disc from drive (use with -A or -B)
    status  - Show drive status (default if no action specified)

Opciones:
  -A, --drive-a  Eject disc from drive A (use with eject action)
  -B, --drive-b  Eject disc from drive B (use with eject action)
  --help         Show this message and exit.
```

### Ejemplos

```bash
cpc drive a              # Select drive A
cpc drive b              # Select drive B
cpc drive eject -A       # Eject disc from drive A
cpc drive eject -B       # Eject disc from drive B
cpc drive status         # Show drive status
cpc drive                # Show drive status (default)
```

## drive a

 Selecciona la unidad A como la unidad activa. El disco debe estar ya insertado usando `cpc disc insert`.

### Uso

```bash
cpc drive a
```

### Ejemplos

#### Select drive A
```bash
cpc drive a
```

<Aside type="caution">
Si la unidad A no tiene un disco insertado, el comando mostrará un error.
</Aside>

<Aside type="tip">
Usa `cpc disc insert -A <file.dsk>` primero para insertar un disco en la unidad A.
</Aside>

</TabItem>

## drive b

Selecciona la unidad B como la unidad activa. El disco debe estar ya insertado usando `cpc disc insert`.

### Uso

```bash

```

### Ejemplos

#### Select drive B
```bash
cpc drive b
```

<Aside type="caution">

</Aside>

<Aside type="tip">
Usa `cpc disc insert <file.dsk> -B ` primero para insertar un disco en la unidad B.
</Aside>

## drive status

Muestra el estado actual de ambas unidades de disquete, mostrando qué archivos DSK están montados y sus rutas.

### Uso

```bash
cpc drive status
# or simply
cpc drive
```

### Ejemplos

#### Mostrar drive status
```bash
cpc drive status
```

#### Default action (no arguments)
```bash
cpc drive
```

<Aside type="tip">
Muestra qué archivos DSK están actualmente montados en las unidades A y B, junto con sus rutas completas y cuál unidad está seleccionada.
</Aside>

<Aside type="note">
La configuración de las unidades se almacena en `~/.config/cpcready/cpcready.toml` bajo la sección `[drive]`.
</Aside>

</TabItem>

<TabItem label="drive eject">

## drive eject

Expulsar el disco DSK actualmente insertado en la unidad A o B, limpiando la configuración de la unidad.

### Uso

```bash
cpc drive eject [OPTIONS]
```

### Opciones

```
  -A, --drive-a  Eject disc from drive A
  -B, --drive-b  Eject disc from drive B
```

### Ejemplos

#### Expulsar from drive A
```bash
cpc drive eject -A
```

#### Expulsar from drive B
```bash
cpc drive eject -B
```

<Aside type="tip">
Usa `cpc drive status` después de expulsar para verificar que la unidad está vacía.
</Aside>

<Aside type="note">
La expulsión elimina la ruta DSK de `~/.config/cpcready/cpcready.toml` para la unidad especificada.
</Aside>

</TabItem>
