## Uso

```bash
Uso: cpc mode [OPTIONS] [VIDEO_MODE]

  Set or display CPC modo de vídeo.

Argumentos:
  VIDEO_MODE  Video mode: 0 (160x200, 16 colors), 
                          1 (320x200, 4 colors), 
                          2 (640x200, 2 colors)

Opciones:
  --help  Mostrar this message and exit.
```

## Ejemplos

### Mostrar modo de vídeo seleccionado
```bash
cpc mode
```

### Set mode 0 (16 colors)
```bash
cpc mode 0
```

### Set mode 1 (4 colors)
```bash
cpc mode 1
```

### Set mode 2 (2 colors)
```bash
cpc mode 2
```


<Aside type="tip">
**Mode 0**: 160x200 pixels, 16 colors  
**Mode 1**: 320x200 pixels, 4 colors  
**Mode 2**: 640x200 pixels, 2 colors
</Aside>
